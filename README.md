# Data Quality Agent with Auto-Fix Suggestions

## Features

- Upload CSV and Parquet files
- YAML-based data quality rules
- Data quality score
- Detect NULL values
- Detect negative values
- Detect invalid emails
- Detect duplicate records
- Auto-fix data issues
- Download cleaned CSV

## Technologies

- Python
- Pandas
- Streamlit
- YAML
- PyArrow

## Run

pip install -r requirements.txt

streamlit run app.py
